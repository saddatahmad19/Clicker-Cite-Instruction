Are you interested in making a click me button in JavaScript?

Lets make this simple counter website in 2 steps!

Disclaimer: The tutorial does not include how to create the instruction set. It only includes the button and the text that increases along with slight CSS stylings.

Step 1

Download the IDE of your choice. An IDE is an Interactive Developement Enviornment (where you program). Some of the most popular IDE's are Microsofts Visual Studio Code and Notepad++.

Click to download an IDE:

If you dont want to download an IDE, you can do this project on an online IDE. Repl.it is a free online IDE that lets you work on and save CS projects in a variety of different languages. If you are going with Repl.it, create a new HTML project and follow allong.

Step 2

Open up your IDE of choice and create a new file. For most applications, you can click File and create a new file. You can name the file whatever you want, but it conventional to name main HTML file index. Regardless of what you name it, it must end with ".html". For example, "index.html".

Step 3

Format your tml file so that you can begin programming your website. Follow the code listed below. 

            
    
        <html>
            <head>
                <meta charset="utf-8">
                <meta name="viewport" content="width=device-width">
                <title>Clicker Website</title>
                <link href="styles.css" rel="stylesheet"/>
            </head>
        
            <body>
                <script src="script.js"></script>
            </body>
        </html>

Make sure you save your work by pressing Ctrl + S or by clicking "File" and then "Save".

Step 4

Inside your body code above the script tag, insert the following,. This is our button and our number. 

            
       
        <button class="clickBtn" onClick="displayCount()">Click Me</button>
        <p id="carrier">0</p>

The "class" attribute is what we use to differentiate our button from other buttons. The onClick refers to the JavaScript funtion we will be implementing later on.

Step 5

Congratulations. You have made the bulk of the website. Press Ctrl + S to save and open up your file. If you are on an IDE, right click the .html file and click "Open in Default Browser"

For Relp.it, you can just run the file. Now you will see your website with a button and the number 0. We're almost done. Now all thats left is to make sure our number increases everytime we lick the button.

Step 6

Now you have the actual button counting part. Create a file called "script.js" that you will upload the code to. 

            
       
                var count = (function() {
                    var counter = 0;
                    return function() {return counter +=1;}
                })();
                
                
                function displayCount() {
                    document.getElementById("carrier").innerHTML = count();
                }

What this does is create a function called count that sets the counter to 0, and every time it is called, it increases counter by 1. The displayCount funtion does exactly what its name implies. It displays the count of the

Step 7

Now its time to make it look nice. This portion is done with css. Create a file called "styles.css" and insert the following. 

            
       
                body {
                    font-family: "Times New Roman";
                }
                
                .clickBtn {
                    font-family: 'Times New Roman', Times, serif;
                    align-items: center;
                    border-radius: 10px;
                    height: 80px;
                    width: 160px;
                    margin-left: 45%;
                    font-size: 30px;
                    background-color: black;
                    color:white;
                }
                
                #carrier {
                    text-align: center;
                    font-size: 30px;
                }
                
                body {
                    background: linear-gradient(to bottom right, lightblue, lightgreen)
                }

Now make sure to save and run.

Step 8

You're done! Congratulations. You build your first HTML website with JavaScript.
